CBETA_TAFxml
============

以「量化語言統計」為目的之簡化版漢文佛典 XML，由 CBETA XML P5 (https://github.com/cbeta-org/xml-p5.git) 經程式轉出。

專案主持人：洪振洲  
轉換程式撰寫：周邦信  
法鼓佛教學院

資料更新紀錄：
- 2019.02.22 以當時最新 CBETA XMP P5，進行本資料集更新，並新增CBETA收錄之卍續藏(X)與嘉興藏(J)資料。
